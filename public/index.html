SONNY SARAVILLO

Medavie Blue Cross

The Ceiling Fan

Description
  • Two button Switch (Reverse and Speed).
  • Speed button, increases speed every time the button pressed. If the button is pressed more than 3x (Max Level of Speed is 3), the fan returns to the “off” or Speed "0" setting.
  • Reverse button, changing the direction of the fan at the current speed setting. Once the direction has been reversed, it remains reversed as the cycle through the speed settings.
  • Default fan speed is "1" and Rotation Type is "Normal".
